# Atividade_EngSoft_II
Criação de códigos monolítico e modularizado para a disciplina de engenharia de software II

# Elementos reutilizáveis:
- A classe Produto pode ser usada em um módulo de vendas, estoque ou financeiro.
- a classe ConsoleUtils pode ser levada para qualquer outro projeto de console java sem nenhuma alteração.
- A classe ProdutoService pode ser injetada em uma API Web ou em uma interface gráfica sem que precisássemos reeescrever a lógica de validação.
